# Moderation themed bot for Tasos's bot jam!
A discord.py botjam with JSON db (stable?)

## Instruction
1. Clone the repo
2. Install poetry
3. [setup](#setup)
4. do `poetry install`
5. do `poetry shell`
6. run `python main.py`
7. enjoy

## setup
create .env file with these values
```bash
DISCORD_TOKEN=token
OWNER_ID=owner
DEPLOY_TYPE=stable
```
